# 2. 모델 클래스 import
- import
   ```console
   > from blog.models import Post
   ```
   - `blog` : 앱 디렉토리
   - `models` = `models.py`
   - `Post` : 모델 클래스

- 메서드 호출
   ```console
   > Post.objects.first()
   <Post: [1 관리자 페이지에서 만듦]>
   > Post.objects.all()
   <QuerySet [<Post: [1 관리자 페이지에서 만듦]>, <Post: [2 두 번째 장고 페이지]>, <Post: [3 서버 지역을 서울로 바꾼 후]>, ...
   ```

- 나가기 : `ctrl + z`
